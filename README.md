## General Info
Automation tests framework created with Selenium and written using Java.

## Technologies Used
* Java - v.17.0.1
* Maven - v.3.6.3
* Selenium WebDriver - v.4.1.4
* TestNG - v.7.6.1
* Lombok - v.1.18.20

## Installation steps
* Clone repository
* mvn clean test
